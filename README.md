# Bellico — премиальный интернет-магазин мебели

Полноценный фронтенд премиального магазина мебели **Bellico**. Интерфейс полностью на русском языке. Сборка не требуется — это статический сайт на чистом HTML/CSS/JavaScript.

## Запуск

Откройте `index.html` в браузере (двойной клик) — всё работает локально, включая корзину, избранное и аккаунт (данные хранятся в `localStorage`).

Для «боевого» режима достаточно отдать папку любым статическим сервером, например:

```bash
npx serve .
# или
python -m http.server 8000
```

## Структура

```
Bellico/
├── index.html          # Главная: герой, ценности, избранный каталог (9 + «Показать ещё»)
├── catalog.html        # Каталог: поиск, фильтры по категориям, сортировка
├── account.html        # Личный кабинет: вход/регистрация + дашборд
├── checkout.html       # Оформление заказа + способы оплаты (демо)
├── info.html           # Info: редактируемые блоки (О компании, Доставка, Оплата, Гарантия)
├── contacts.html       # Контакты: реквизиты-плейсхолдеры + форма обратной связи
├── privacy.html        # Политика конфиденциальности (юридический шаблон)
├── css/
│   └── styles.css      # Дизайн-система и все компоненты
└── js/
    ├── data.js         # Каталог товаров + генератор SVG-изображений
    ├── state.js        # Хранилище: корзина, избранное, авторизация, заказы (localStorage)
    ├── components.js    # Шапка, подвал, корзина-drawer, модальное окно, тосты, поиск, меню
    ├── shop.js         # Карточки, сетка, фильтры/сортировка, модалка товара + варианты
    ├── account.js      # Авторизация и личный кабинет
    ├── checkout.js     # Чекаут и заглушки оплаты
    └── app.js          # Бутстрап страниц и форма контактов
```

## Ключевые возможности

- **Каталог**: сетка карточек, наведение → зум + цена + «Подробнее».
- **Модалка товара** (без отдельных страниц): галерея, описание, характеристики, цена, выбор количества, «В корзину» и «Купить сейчас».
- **«Другие варианты»**: мгновенное переключение модификаций (ткань/цвет/материал) без перезагрузки.
- **Корзина-drawer**: выезжает справа, управление количеством, удаление, итог.
- **Аккаунт**: регистрация (Имя, Email, Телефон, Пароль), вход, дашборд (Профиль, История заказов, Избранное, Адреса доставки, Настройки).
- **Чекаут**: контактные данные, доставка, оплата.
- **Избранное, поиск, фильтры, сортировка**, плавные анимации, доступность, адаптив (десктоп + мобайл).

## Где заменить контент

- **Товары**: массив `RAW` в `js/data.js`. Чтобы подключить реальные фото, добавьте варианту поле `gallery: ['url1', 'url2', 'url3']` (заменит сгенерированные SVG-плейсхолдеры).
- **Info**: блоки помечены `<!-- EDITABLE BLOCK -->` в `info.html`.
- **Контакты**: телефон/email/адрес помечены `<!-- EDITABLE -->` в `contacts.html`.
- **Политика**: текст в квадратных скобках `[ ... ]` в `privacy.html`.

## Подключение оплаты

Оплата сейчас работает в демо-режиме. Точка интеграции — функция `createPayment(method, order)` в `js/checkout.js`: реализуйте вызов эквайринга / СБП / ЮKassa и редирект на платёжную страницу. UI, объект заказа и сводка уже подготовлены.

> ⚠️ Авторизация в `js/state.js` — клиентская демо-реализация на `localStorage` (пароли не шифруются). Для продакшена замените на реальный бэкенд/API аутентификации.
